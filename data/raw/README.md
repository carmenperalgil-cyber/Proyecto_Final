# Raw Data

Los datos originales no se incluyen en este repositorio debido a su tamaño.

## Fuente

- Inside Airbnb: http://insideairbnb.com
- Eurostat: https://ec.europa.eu/eurostat

## Instrucciones

Descargar los datasets correspondientes a:

- Madrid
- Mallorca
- Valencia
- Sevilla
- Milán

Y colocarlos en esta carpeta con los siguientes nombres:

- madrid_listings.csv
- mallorca_listings.csv
- valencia_listings.csv
- sevilla_listings.csv
- milan_listings.csv
- turismo.csv
